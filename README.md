## GoogVoiceBank-UTAU
### 简介
+ 谷歌娘声库，UTAU版本
+ 之前搞的DeepVocal版本因为软件问题放弃了，现在用回UTAU
+ 还是基于 `BV1V54y1y7Eb` 制作的
+ 主要重调了固定发音参数，修复一些缺少的发音
+ 先计划把基础部分做好，之后有时间可能会完善扩张整音和单独元辅音
+ 做好后会有示例工程
